# 合成大西瓜Unity复刻版

这是一个复刻“合成大西瓜”小游戏的Unity项目，本人为大二学生，在家闲着无聊，本项目现已实现了大部分玩法与规则，要是有觉得我哪里写的不好或者有更好的解决办法的话，希望能指出我的问题。

## 开发环境

Unity2017.4.34f1  
vscode  

## 文件位置

场景文件储存在 `./Assets/Scenes` 文件夹中  
`Index`为开始游戏场景  
`Game`为游戏场景  
`Over`为游戏结束场景
