# 合成大西瓜Unity复刻版

这是一个使用Unity复刻“合成大西瓜”的简单项目  
寒假在家没事干就想随便做一下，现在已经基本完成项目的编写

## 开发环境

Unity2017.4.34f1  
vscode  

## 文件位置

场景文件储存在 `./Assets/Scenes` 文件夹中  
`Index`为开始游戏场景  
`Game`为游戏场景  
`Over`为游戏结束场景
